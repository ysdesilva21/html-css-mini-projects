# 🚀 Responsive Navbar & Hero Section

A modern, responsive **Navbar + Hero section** built using **HTML, CSS, and minimal JavaScript**.  
This project is part of my **HTML & CSS Mastery series**, focusing on layout fundamentals, spacing systems, and responsive design.

---

## 📌 Overview

This project demonstrates how to build a **production-style landing section** without frameworks.

### Core focus areas:

- Flexbox for layout and alignment
- CSS Grid for hero structure
- Responsive design (mobile → desktop)
- Spacing system using CSS variables
- Clean UI with reusable components

---

## 🧠 What I Learned

- Structuring layouts using **Flexbox vs Grid**
- Building a **mobile-first responsive navbar**
- Handling **hamburger menu without heavy JS**
- Creating a **scalable design system** with CSS variables
- Managing **spacing, typography, and consistency**
- Adding subtle **animations and transitions**
- Improving **accessibility (ARIA, focus states, reduced motion)**

---

## ⚙️ Tech Stack

- HTML5 (Semantic structure)
- CSS3
  - Flexbox
  - Grid
  - CSS Variables
  - Media Queries
  - Animations
- Vanilla JavaScript (for menu interaction)

---

## ✨ Features

- ✅ Fixed, blurred navbar with smooth entry animation
- ✅ Fully responsive hamburger menu
- ✅ Click-outside + auto-close navigation behavior
- ✅ Hero section with grid layout
- ✅ Reusable button components (primary, secondary, CTA)
- ✅ Design system (colors, spacing, typography)
- ✅ Gradient text + subtle UI animations
- ✅ Accessibility support:
  - Focus states
  - Reduced motion
  - ARIA labels

---

## 📱 Responsive Behavior

| Device Size     | Behavior                        |
| --------------- | ------------------------------- |
| Mobile (<768px) | Hamburger menu, stacked layout  |
| Tablet          | Adjusted spacing, vertical hero |
| Desktop         | Full navbar + 2-column hero     |
| Large screens   | Increased spacing and width     |
